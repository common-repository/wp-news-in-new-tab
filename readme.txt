== Description ==
WordPress news plugin allows WordPress website administrators to open WordPress news in new tab in place of opening the news within the WordPress website in default design. This plugin is developed using jQuery technology. This also opens news RSS feed in new tab.

Contact <a href="http://ifourtechnolab.com">custom WordPress website development company</a>  for any assistance.

== Installation ==
1. Upload wp-news-in-new-tab.zip file in you `/wp-content/plugins/`  directory and unzip it.
2. Activate the plugin through the '\Plugins\' menu in WordPress.
